## Project Description

MeetMode is an AI-powered networking and event discovery platform that connects users based on skills, interests, and events.

It includes:
- Dashboard system
- Friends & chat system
- Event discovery
- AI-based matching
- QR networking
- Notifications system

Built using React frontend + FastAPI backend.

---

## Approach

- Built modular FastAPI backend
- Designed React dashboard UI
- Implemented event + social flow
- Added AI-based matching logic
- Used Valkey for fast data storage
- Focused on scalable architecture
