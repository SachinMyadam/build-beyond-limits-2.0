## Project Description
MeetMode is an AI-powered networking platform built using FastAPI and React. It provides user management, friends, events, chat, QR networking, notifications, AI memory, AI search, and AI-powered event recommendations using Valkey.

## Approach
Built a modular FastAPI backend and React frontend. Integrated AI memory and recommendations while using Valkey for fast key-value storage and retrieval.
